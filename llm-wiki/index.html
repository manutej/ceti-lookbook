This is the shift. For ten years, PKM tools have asked humans to write notes for their future selves. Obsidian, Roam, Logseq, Notion — they all stop at the page. The agent could read your vault, but it couldn't grow it. Every conversation started from zero.

Now the agent has hands. It reads your sources, summarizes them into wiki pages, draws the links you forgot, files them under the right concept, and keeps the graph internally consistent. You curate. It compiles. The wiki is the codebase. The LLM is the programmer.

The Karpathy framing

An idea file, gone viral.

On April 4, 2026, Andrej Karpathy posted a one-page Gist describing a pattern for personal LLM-maintained knowledge bases. Within two weeks: 5,000 stars, dozens of open-source re-implementations.

"Obsidian is the IDE; the LLM is the programmer; the wiki is the codebase."
Andrej Karpathy · llm-wiki.md · April 2026

Karpathy frames it as a compilation problem. Raw sources — articles, podcasts, papers, screenshots, your own messy thoughts — are the source code. The wiki is the compiled artifact: cross-referenced, summarized, queryable. You don't read the source every time you want an answer; you read the compiled wiki. The LLM does the compile pass, repeatedly, as new sources land.

Karpathy's own vault, built this way over a few months, became a 400,000-word personal textbook on the topics he was thinking about. Not written by him. Maintained for him.

The four ingredients

Building blocks of an agentic PKM.

Strip away the tooling debates. Four primitives are all that's needed: a substrate the AI can read, a structure it can traverse, an agent with write access, and a way to ask your past self questions.

01 · Substrate

Notes — markdown is the universal format.

Plain text files with [[wikilinks]] are the only substrate that survives every tooling change of the next decade. Agents can read them. Humans can read them. Git can version them. No vendor owns them.

02 · Structure

Links the AI can traverse.

Backlinks turn a folder of notes into a graph. The agent walks edges to find context — far cheaper and more accurate than vector search alone.

03 · Agency

Agents with read AND write.

Read-only is dead-on-arrival. The shift is granting the agent permission to file, link, summarize, and refactor your vault — under your review.

04 · Recall

Queries — questions you ask your past self.

"What did I conclude about RAG in Q3?" "Show me every decision I tagged 'reversible.'" "Draft a synthesis of everything I've read about agentic patterns this year." The vault becomes a thinking partner that remembers what you do not.

Three ways to ship this today

Real implementations, shipping now.

None of this is theoretical. Three stacks have crystallized in the months since Karpathy's gist — each trades polish, ownership, and agency in different proportions.

Path A · Plug-and-play

Obsidian + Smart Connections

A community plugin that embeds every note, retrieves on-demand, and now writes back via the new MCP bridge. Your vault stays on disk; the AI does the indexing. Lowest setup cost; best for solo knowledge workers.

Local-first ~20 min setup
Path B · Team-friendly

Notion + Claude MCP

Notion's MCP server lets Claude read, edit, and create pages across your workspace. Best when the knowledge graph is shared — research teams, founders with assistants, consultants with client knowledge bases.

Cloud-first Multi-user
Path C · Power-user

Markdown folder + Claude Code

A git-versioned directory of .md files, with Claude Code as the agent. Maximum control, maximum customization, zero vendor lock-in. The path Karpathy himself ships on.

Self-hosted Git-versioned

A day in the agentic vault

What it feels like in practice.

Not a productivity system. A loop. Capture is fast; query is conversational; the cleanup happens while you sleep.

01
Morning · 07:42

Capture

Voice memo on your walk, three links from Twitter, a screenshot from a paper. All dropped raw into /sources.

02
Midday · 12:18

Query

"What did I think about long-context vs. RAG in March?" The agent walks the graph, surfaces three notes, and shows the contradictions you'd forgotten.

03
Afternoon · 15:30

Co-draft

You ask for a 500-word essay drawn from your notes on agentic PKM. The agent drafts; you edit; the edits become a new wiki page, auto-linked.

04
Evening · 22:05

Compile

Overnight, the agent processes today's sources, files them, draws new edges, and pushes a single git commit. You wake up to a vault that's 6 notes smarter.
