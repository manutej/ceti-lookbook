About

The CETI Lookbook is a curated wall of eighteen explainers on the 2026 AI / agent landscape — each one rendered in a different visual language to demonstrate what's possible when the design system is the question, not the answer.

Every page is single-file HTML, under 110KB, mobile-first, accessibility-checked, and built with the shadcn-motion Claude Code skill (which lives in this repo at /skills/shadcn-motion).
