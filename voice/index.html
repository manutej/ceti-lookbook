The latency budget

Sub-second, or it isn't a conversation.

Humans hand off conversational turns in roughly 200ms. Anything above 800ms and the agent has lost the room.

STT · 150ms
LLM TTFT · 400ms

≈ 800ms end-to-end

In natural human conversation, the gap between one speaker finishing and the next starting averages 200–500 ms. Beyond 800 ms, people unconsciously slow their cadence, over-explain, or start to repeat themselves — the unmistakable feeling of talking to a machine.

The 2026 stack hits the budget by streaming: STT emits partial transcripts before the user finishes, the LLM begins generating before STT completes, and the TTS starts speaking before generation ends. Combined streaming can shave 300–600 ms off a naive pipeline. The LLM still owns roughly half the wall clock — pick the wrong model and nothing else matters.

The stack

Four layers, composable.

Most production agents wire these four boxes together. The orchestration layer is what turns it from a demo into a phone number that answers.

STT

Speech → Text

Streaming transcription with word-level timestamps and partial hypotheses. Strong models now hit 150 ms first-token latency in 70+ languages.

The glue that ships

Wires STT → LLM → TTS over a single duplex socket, handles barge-in, dial-tone integration, retries, tool-calling, and telemetry. Without it, you have a demo — not a phone number.

Minimal agent

Twelve lines, a phone call.

An OpenAI Realtime session over WebSocket. The model speaks back as audio chunks on the same connection; the client just plays them.

voice-agent.ts 
import WebSocket from "ws";

const ws = new WebSocket(
  "wss://api.openai.com/v1/realtime?model=gpt-4o-realtime",
  { headers: { Authorization: `Bearer ${process.env.OPENAI_API_KEY}` } }
);

ws.on("open", () => ws.send(JSON.stringify({
  type: "session.update",
  session: { voice: "verse", modalities: ["audio", "text"] }
})));

ws.on("message", (buf) => {
  const e = JSON.parse(buf.toString());
  if (e.type === "response.audio.delta") play(e.delta); // stream to speaker
});
